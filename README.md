### Node

```
node myfile.js
```


### Typescript


Install ts node

```
# Locally in your project.
npm install -D typescript
npm install -D ts-node

# Or globally with TypeScript.
npm install -g typescript
npm install -g ts-node
```

```
ts-node myfile.ts
```
